# Redis

* Redis是一个开源，先进的key-value存储，并用于构建高性能，可扩展的Web应用程序的完美解决方案。
* Redis从它的许多竞争继承来的三个主要特点：
* Redis数据库完全在内存中，使用磁盘仅用于持久性。相比许多键值数据存储，Redis拥有一套较为丰富的数据类型。
* Redis可以将数据复制到任意数量的从服务器。
